# API de Pessoas (Flask)

API REST desenvolvida com Flask para gerenciamento de pessoas.

## 🚀 Funcionalidades
- Criar pessoa
- Listar pessoas
- Buscar por nome
- Atualizar pessoa
- Remover pessoa

## 🛠 Tecnologias
- Python
- Flask
- JSON (persistência)

## 📌 Endpoints

GET /pessoas  
POST /pessoas  
GET /pessoas/{nome}  
PUT /pessoas/{nome}  
DELETE /pessoas/{nome}

## ▶️ Como rodar

```bash
pip install flask
python app.py
